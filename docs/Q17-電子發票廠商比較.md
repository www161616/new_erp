# Q17 電子發票廠商比較（2026-04-23 掃描）

> **目的**：會計師諮詢前先自行盤點 5 家主要電子發票服務商，把 §5、§9 的預算題先縮小範圍。
> **關聯**：[[Q17-會計師諮詢清單]] §5（電子發票 API 串接）、§9（成本預估）

---

## TL;DR
- **首推：綠界 ECPay** — 2026 新戶方案首年 0 元可開 20,000 張，續年費最便宜，API/文件成熟
- **次推：ezPay** — 後台 UI 對非技術店主最親切，但成本比綠界高 1.5~2 倍
- **跳過：藍新 / 關貿 / 永豐** — 藍新須綁金流，另外兩家對 100 家小加盟規模不經濟

## 對「100 家加盟店 × 每店 500–2000 張/月」的試算

假設中位數每店 1,000 張/月 = 12,000 張/年：

| 廠商/方案 | 單店首年 | 單店續年 | **100 家總計（首年）** | **100 家總計（續年）** |
|---|---|---|---|---|
| 綠界 20,000 張方案 | $9,600 | $6,000 | **$960,000** | **$600,000** |
| 綠界 5,000 張方案（不夠）| — | — | — | — |
| ezPay 菁英 3,000 張/月 | $12,000 | $12,000 | **$1,200,000** | **$1,200,000** |

**⚠️ 若抓得到 2026/6/30 前的綠界新戶 0 元優惠，100 家首年可省 $600,000**（但要 100 家在期限前都送件並完成 KYC）。

試算僅含發票服務費本身，不含：簡訊通知、超商列印中獎發票、會計師顧問費、加盟店培訓費、系統整合開發費。

## 每家重點

### 綠界 ECPay（首推）
- 2026 新方案：新戶首年 0 元、20,000 張；設定費 $3,600 僅收一次
- SDK 多、文件公開、社群範例多、工程整合成本低
- 續年費在三家中最便宜（$6,000 / 20,000 張）
- 簡訊 $1.5/封、超商列印中獎 $20/張，採月結
- **未知項**：多統編 / 多賣家計費結構（官網沒寫，必問業務）

### ezPay 簡單付（次推 / B 方案）
- 屬藍新集團子公司
- 後台 UI 親民，適合非技術店主自助查帳
- 年繳方案三階：基本 $6,000 / 菁英 $12,000 / 尊榮 $18,000
- 計費制「精算型」約 $1.2/張，適合低頻店
- **未知項**：多統編支援、API 測試環境、作廢折讓是否免費

### 藍新 NewebPay（跳過）
- 主力是金流，發票為配套服務
- 若 v1 不使用藍新金流，議價力低、價格偏高
- 文件部分需登入後台才看得到

### 永豐金 SinoPac（跳過）
- 銀行體系，適合已有永豐企金往來的企業
- 對 100 家小加盟 KYC 成本極高、上架慢（3–6 週/店）
- API 文件封閉、社群範例少

### 關貿 Trade-Van（跳過）
- 企業級、多稅籍/B2B EDI 強項，政府與大型製造業常用
- 對「每店月 500–2000 張」殺雞用牛刀
- 上架流程長（4–8 週）、需專案經理介入

## 對「100 家加盟店」場景的建議

1. **多統編必選原生支援**：每家加盟店統編不同，**不可用總部單一統編代開**（踩稅法紅線，詳 Q17-會計師諮詢清單 §4）。首選綠界，詢問「主帳號 + 子商店」模式。
2. **計費分攤**：主帳號集中議價、子商店計費分開開立三聯式給各加盟店（總部取得統一報價、帳單給各店支付）。
3. **培訓成本**：店主多為婆婆媽媽型，後台 UI 友善度 ezPay > 綠界 > 藍新 > 永豐/關貿。可考慮**綠界為主 + 對 UI 敏感的店讓它們選 ezPay** 雙軌策略（但會增加維運複雜度）。
4. **降級/斷線**：無論選誰，**總部 ERP 必須自建發票佇列**（廠商 API down 時先記錄、恢復後補送）。財政部要求 36 小時內上傳，不能把壓力推給店家。
5. **搶新戶優惠**：2026/6/30 是綠界新戶 0 元的關鍵日，**若決定走綠界，需在此日期前完成 100 家送件**。

## 還不確定、需會計師/業務確認的點

對應 [[Q17-會計師諮詢清單]] §5（電子發票 API 串接）、§9（成本預估）：

- [ ] **綠界多統編計費結構**：100 個統編是 100 份合約？還是主帳號 + 子商店？
- [ ] **綠界「新戶」定義**：總部或加盟店之前用過綠界金流者是否算新戶？
- [ ] **綠界量大折扣**：100 店 × 12,000 張 = 120 萬張/年，有無企業整批價
- [ ] **ezPay 多統編支援**（官網未公開）
- [ ] **SLA / 賠償條款**：API 中斷、資料外洩、漏開立的責任歸屬
- [ ] **7 年稅法保存責任**：廠商備份 vs 總部 ERP 自存
- [ ] **關貿/永豐 B 方案報價**：若未來 v2 要跨境或 B2B EDI，能否另案評估（非 v1 必做）
