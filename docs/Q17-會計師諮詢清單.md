# Q17 發票政策 — 會計師諮詢清單

> **目的**：帶此份清單去請會計師諮詢，一次釐清所有發票 / 稅務疑問，避免來回多次。
> **關聯**：[PRD-訂單取貨模組 §13 Q17](PRD-訂單取貨模組.md)（⚠️ Open，上線前必決）

---

## 📌 背景說明（請會計師先讀）

- **業態**：團購店連鎖、**100 家加盟店（非直營）**
- **顧客流程**：LINE 社群下訂 → 到門市取貨 → 現場結帳（v1 全面現金）
- **現況**：v1 **暫定不開發票**，僅在取貨單列明細
- **風險認知**：依稅法，月營業額 > 20 萬須強制開立統一發票、未開立屬違法
- **上線目標**：v1 預計 [____年__月] 開始 pilot → 需在此之前完成合規

## 🟡 待請教問題

### 1. 稅籍 / 納稅狀態

- [ ] **1.1** 集團總部目前是否已登記為營業人？稅籍狀態為？
- [ ] **1.2** 100 家加盟店各自的稅籍狀態？（獨立登記 vs 依附總部？）
- [ ] **1.3** 加盟合約中稅務分工條款有無？（總部代開 vs 各店自開）
- [ ] **1.4** 每家加盟店是否都有自己的統一編號？

### 2. 營業額 / 20 萬門檻

- [ ] **2.1** 月營業額 20 萬門檻：是**集團總額**還是**各店個別**計算？
- [ ] **2.2** 各加盟店的預估月營業額分佈如何？超過門檻的比例估計？
- [ ] **2.3** 若只有部分店超標，合規做法是？（全集團統一 vs 只超標店開）
- [ ] **2.4** 新加盟店上線初期營業額未達標、後來達標的銜接方式？

### 3. 發票類型選擇

- [ ] **3.1** 「營業人統一發票」vs「小規模營業人免開」哪種適合本業態？
- [ ] **3.2** 若採營業人發票，**電子發票**還是**紙本二聯/三聯**？
- [ ] **3.3** 電子發票 B2C 是否已是主流？顧客接受度 / 中獎領獎流程？
- [ ] **3.4** B2B 情境（極少、員工團購）需要三聯式嗎？
- [ ] **3.5** 免用發票收據（收銀機發票）能否替代？

### 4. 開立責任分工（加盟店特殊性）

- [ ] **4.1** 由**總部統一開立**的可行性？（顧客實際交易在加盟店）
- [ ] **4.2** 各**加盟店各自開立**時，總部系統要怎麼 enforce 合規？
- [ ] **4.3** 若總部只負責收集營業額、不開票，法律風險歸屬在總部還是加盟店？
- [ ] **4.4** 加盟金 / 權利金 / 採購分潤的發票流向是？

### 5. 電子發票 API 串接

- [ ] **5.1** 建議直接串接**財政部電子發票平台**、還是經第三方代理？
  - 候選廠商：綠界 / 藍新 / ezPay / 永豐 / 關貿 / 資通電腦 / ...
  - 會計師有推薦的嗎？
- [ ] **5.2** 導入費用區間（一次性設定 + 月費 + 每張手續費）？
- [ ] **5.3** 開立失敗時的補救機制（系統當機、雲端中斷）？
- [ ] **5.4** 折讓單 / 作廢單的 API 支援？

### 6. 員工團購（§7.10）發票

- [ ] **6.1** 員工走員工價（75% 折）+ 月結扣薪，是否需要開發票？
- [ ] **6.2** 若需要，月結時開**一張彙總發票** vs 每次各開一張？
- [ ] **6.3** 員工折扣涉稅務嗎？（是否被認定為實物福利、需列入薪資課稅？）

### 7. 退貨 / 折讓（§7.11）

- [ ] **7.1** 當天內退貨、原發票已開 → 走**發票作廢** vs **折讓單**哪個？
- [ ] **7.2** 現金退費的帳務憑證處理？
- [ ] **7.3** 退貨倉（報廢）vs 瑕疵品倉（再處理）的稅務成本扣抵如何處理？

### 8. 合規時程 / 風險

- [ ] **8.1** **v1 若暫不開票，最長多久必須完成合規？**
- [ ] **8.2** 稅捐稽徵機關查核風險範圍？罰則（罰鍰 + 補稅）試算？
- [ ] **8.3** 過渡期可採的「手工收據」合法性？（目前 v1 做法）
- [ ] **8.4** 建議上線前「最少」完成哪些合規動作才敢開門營業？

### 9. 成本 / 預算預估

- [ ] **9.1** 會計師費用（一次性設定 + 每月顧問 + 年度報稅）？
- [ ] **9.2** 電子發票廠商費用（對應 5.2）？
- [ ] **9.3** 加盟店培訓 / 設備費用（發票機、印表機、雲端）？
- [ ] **9.4** 建議全集團導入預算區間？（低 / 中 / 高方案）
